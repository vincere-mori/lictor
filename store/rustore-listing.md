# RuStore - карточка приложения Lictor

## Название (до 50)
Lictor: напоминания, которые дожимают

## Краткое описание (до 80)
Жёсткие напоминания со ступенчатой эскалацией. Давит, пока не сделаешь.

## Полное описание (до 4000)
Lictor - менеджер задач для тех, кому вежливые напоминалки не помогают. Он не уговаривает. Он дожимает.

В основе лестница давления из трёх тиров:

MONEO - тихий толчок. Просто чтобы не забыл.
INSTO - настойчиво. Возвращается снова и снова, пока не отреагируешь.
COGO - полноэкранная тревога. Отложить можно только удержанием.

Каждой задаче ты сам назначаешь, насколько жёстко её вести.

Адаптивный «Мозг»
Lictor смотрит, когда ты реально доводишь дела до конца, и ставит туда задачи без точного времени. Серия дней подряд, статистика за неделю, граф связей между задачами и группами.

Что внутри
- Полностью офлайн. Данные хранятся только на устройстве.
- Без рекламы, без трекеров, без обязательных аккаунтов.
- Естественный ввод: «позвонить маме завтра 18:00 жёстко».
- Группы задач, быстрые чипы времени, налог за каждый перенос.
- Четыре темы: две тёмные, две светлые. Чернила по бумаге.
- Стоические строки под тон задачи и момента.

Для кого
Для студентов, фрилансеров и всех, кто срывает сроки не от лени, а потому что обычные напоминания слишком мягкие.

sine mora. Без промедления.

## Что нового (v1.0.2)
Граф связей задач переработан в стиль нейронов: свечение узлов, подсветка соседей при наведении. В редакторе задачи Enter сохраняет, Escape закрывает. Ускорения и мелкие правки.

## Сбор данных
Приложение не собирает персональные данные. Все задачи хранятся локально на устройстве. Опциональные веб-пуши используют только подписку браузера и работают, если пользователь сам их включил.
